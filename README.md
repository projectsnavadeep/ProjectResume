## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

The application will be available at `http://localhost:5173`

## Usage

1. **Enter Your Details** - Fill in the form with your personal information, education, skills, projects, experience, certifications, and achievements
2. **Choose a Template** - Select from Minimal, Modern, or Professional Corporate styles
3. **Preview Live** - Watch your resume update in real-time
4. **Download** - Export your resume in PDF, DOCX, or HTML format

## Supported Formats

- **PDF** - Print-ready A4 format, perfect for sharing
- **DOCX** - Editable Microsoft Word format
- **HTML** - Web-friendly format, optimized for printing
